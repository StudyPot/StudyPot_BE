# 03 유저 플로우

## Host
1. Create group.
2. Submit own onboarding.
3. Share invite link.
4. Monitor onboarding status.
5. Start study.
6. Check weekly progress and AI feedback.

## Member
1. Join by invite.
2. Submit onboarding.
3. View current week.
4. Complete todos or submit incomplete reason.
5. Chat with AI team leader.
6. Read retrospective feedback.

## Late Joiner
1. Join active group.
2. Submit onboarding.
3. Enter current week.
4. Future AI feedback may use late onboarding context.
